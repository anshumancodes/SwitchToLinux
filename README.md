# SwitchToLinux
SwitchToLInux project aims to help people switching linux as their primary Os.
 It helps people understand the advantages of Linux , Which disto to choose according to the users need and much more!

## Tech
This project is a simple html css(or tailwind) js based project

-[tailwind](https://tailwindcss.com/docs/installation)

## Contribution
-SwitchToLinux is a open-source project and you can contribute to improve it.

```bash

```
