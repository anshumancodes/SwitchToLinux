# SwitchToLinux

SwitchToLinux is an open-source project designed to assist people in transitioning smoothly to Linux with our guides.


## clone it 
```bash

git clone https://github.com/anshumancodes/SwitchToLinux.git

cd SwitchToLinux

npm i


```
