Introduction to linux

- We all have used windows operating system as kids , or lets say at-least heard about it? right we basically know what a operating system do right :like it let you play your favourite game or a let you draw on ms paint . So on a bit technical level Operating systems bring together hardware and software to make sure its usable i.e; is users can perform all the tasks they desire alright! now Linux is similarly a operating system based on UNIX (yet another operating system lets forgot about it )that originated back in 90s(1990s) and you wont believe but Linux is everywhere! guess what your mobile runs on a Linux based OS!your car's smart system (yeah the one can let you pick calls ,got a nice touch screen and all that ) runs on Linux!!!

Now Linux operating system comprises several different pieces:

  • Bootloader :A bootloader is a program or lets say software piece that runs first whenever you turn on your compute, like you see the windows or apple icon whenever you turn your computer on? thats when the bootloader is at work i.e; it loads the os into the ram after locating the operating system kernel, which is the core of the operating system.
  • Kernel: one piece of the whole that is actually called ‘Linux’. The kernel is the core of the system and manages the CPU, memory, and other connected devices. The kernel is the lowest level of the OS.
